# 개인정보처리방침 (FTP 업로드)

파일: [`privacy.html`](privacy.html)

## 업로드 전 수정 (노란색 placeholder)

| 항목 | 예시 |
|------|------|
| 시행일 | `2026-07-17` |
| 운영자/상호 | 본인 또는 사업자명 |
| 문의 이메일 | 실제 수신 가능한 메일 |

HTML에서 `YYYY-MM-DD`, `운영자/상호명`, `privacy@example.com` 을 검색해 바꾸세요.

## FTP

웹 루트에 올린 뒤 Play Console / 앱에 아래처럼 연결하면 됩니다.

```text
https://your-domain.com/privacy.html
```

권장: `privacy.html` 또는 `privacy/index.html`
